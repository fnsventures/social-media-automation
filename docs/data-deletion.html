User Data Deletion Instructions

Last updated: June 10, 2026

This app is used internally to publish content to a single Facebook Page, Instagram Business account, and YouTube channel. To remove your data from this app:

Steps to delete data

  1. Revoke app access in Facebook: Settings → Apps and Websites, remove this app.
  2. Revoke YouTube access in Google: myaccount.google.com → Security → Third-party access, remove the OAuth app.
  3. Remove stored tokens from GitHub: delete META_* and YOUTUBE_* repository secrets.
  4. Delete local .env credentials from your computer.
  5. Optionally delete published post YAML files from the repository if you want content removed from version history (contact repo admin).
